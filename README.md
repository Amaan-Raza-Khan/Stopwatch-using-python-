# ⏱ Professional Stopwatch (Python + Tkinter)

A modern and professional Stopwatch Desktop Application built with **Python** and **Tkinter**.

## 🚀 Features

- ▶ Start Stopwatch
- ⏸ Pause / Resume
- 🔄 Reset Timer
- 🏁 Lap Recording
- 🗑 Clear All Laps
- 📄 Export Laps to Text File
- 💾 Save As Support
- 📋 Scrollable Lap History
- 🎨 Clean Modern Dark UI

## ▶ How to Run

Clone the repository

```bash

```

Go inside folder

```bash
cd Stopwatch-using-python
```

Run

```bash
python gui.py
```
